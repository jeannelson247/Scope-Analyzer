Synthetic V/I/di-dt examples for Scope Studio
================================================

These files are deterministic educational data, not lab data.
L = 0.000166 H (166 uH)
R = 0.012 ohm
Model: dI/dt = (V_drive(t) - R*I) / L
Derived voltage: L*dI/dt

Surface files use long format:
x = time_ms, y = drive_voltage_V, z = selected value, c = color-encoded companion value.
