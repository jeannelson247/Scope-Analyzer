# Scope Analyzer Classic

Stable native PySide reference build from `app.py`. Keep this for lab use
while the web app catches up. Do not rewrite `app.py` for the web UI.

Developer build:

```bash
cd ~/Desktop/scope_studio03
source venv/bin/activate
pip install pyinstaller
./scope_analyzer/classic/build.sh
open dist/ScopeAnalyzerClassic.app
```
